# 📄 Resume + Job Match AI Tool

An AI-powered web application that compares a resume with a job description and provides a match score along with actionable improvement suggestions.

🔗 **Live App:** https://resume-match-ai.streamlit.app  
📂 **GitHub Repo:** https://github.com/11-Chandangowda/resume-match-ai  

---

## 🚀 Features

✅ Upload resume (PDF)  
✅ Paste job description  
✅ Match score calculation using NLP  
✅ Missing skills detection  
✅ ATS-friendly improvement suggestions  
✅ Fast and lightweight UI (Streamlit)

---

## 🧠 Tech Stack

- **Python**
- **Streamlit**
- **Scikit-learn (TF-IDF + Cosine Similarity)**
- **pdfplumber (PDF parsing)**
- **Regex (text preprocessing)**

---

## ⚙️ How It Works

1️⃣ Extracts text from uploaded resume  
2️⃣ Cleans and processes both texts  
3️⃣ Uses TF-IDF vectorization  
4️⃣ Calculates similarity score  
5️⃣ Identifies missing keywords  
6️⃣ Provides actionable feedback
